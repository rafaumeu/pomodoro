## Overview

A **Pomodoro Technique** timer application with integrated task tracking. Built with **React** and **TypeScript**, featuring a clean interface for focused work sessions with break management.

## Features

- Configurable Pomodoro timer (25/5/15 min)
- Task list with completion tracking
- Visual countdown with progress indicator
- Audio notifications for session transitions
- Automatic session cycling

## Tech Stack

| Technology | Purpose |
|---|---|
| **React** | UI library |
| **TypeScript** | Type safety |
| **Styled Components** | Styling |

## Getting Started

```bash
git clone https://github.com/rafaumeu/pomodoro.git
cd pomodoro
npm install
npm run dev
```
